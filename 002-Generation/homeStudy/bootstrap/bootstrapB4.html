Conteiner

Os conteiners são os elementos de Layout mais básicos no bootstrap e são usados para conter, preencher e as vezes centralizar o conteudo dentro deles.

Por padrão o bootstrap estabelece os pontos de quuebra (media query) para alguns tamanhos de tela/janelas(são responsivos).
