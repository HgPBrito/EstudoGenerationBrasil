Reboot (reinialização)

A reinicialização baseia-se apenas na padronização de estilo para alguns elementos do HTML, fazendo o uso de seletores CSS pelo nome de tags e atributos.


Página padrão e fontes nativas

Os elementos <html> e <body> são modificados para melhor exibição da página padrão.

Parágrafos

O elemento HTML <p> (ou elemento HTML de parágrafo) representa um parágrafo do texto.


Listas

O elemento HTML <ul> (ou elemento HTML de lista desordenada) representa uma lista de itens sem ordem rígida, isto é, uma coleção de itens que não trazem uma ordenação numérica e as suas posições.

  • Abacaxi
    • Pérola
    • Imperial
    • Ajubá
  • Banana
  • Graviola

O elemento HTML <ol> (ou elemento HTML de lista ordenada) representa uma lista de itens em ordem rígida, isto é, uma coleção de itens que trazem uma ordenação numérica e as suas posições.

  1. Abacaxi
    1. Pérola
    2. Imperial
    3. Ajubá
  2. Banana
  3. Graviola

O elemento HTML <dl> (ou elemento HTML de lista de definições) representa uma lista de pares de termos e descrições.

Frutas
Abacaxi, Banana e Graviola
Legumes
Abobrinha, Chuchu e Batata

Textos pré-formatados

O elemento HTML <pre> representa o texto pré-formatado que deve ser aprensentado exatamente como está escrito no arquivo HTML. 

        ***************************
        **  TEXTO PRÉ-FORMATADO  **
        ***************************
