COMANDOS DO NAVEGADOR

 CTRL+O  -> para abrir arquivos . 

 F12  -> para abrir a janela das ferramentas de desenvolvedor.


TAGS HTML

<meta charset="UTF-8">  -> define o padrão de caractéres do arquivo para o navegador. 

<h1> exemplo </h1>  -> marca exemplo como título principal.

<p>exemplo</p>  -> marca exemplo como parágrafo.

<big> exemplo </big>  -> aumenta o tamanho da letra do exemplo.

<br>  -> quebra linha.

<hr>  -> cria uma linha continua.

<a href="link">exemplo</a>  -> âncora para links.

<script>código JavaScript</script>  -> tag para o navegador interpretar código JS.

<input/>  -> cria uma caixa de entrada

<button>exemplo</button> -> cria um botão escrito exemplo
